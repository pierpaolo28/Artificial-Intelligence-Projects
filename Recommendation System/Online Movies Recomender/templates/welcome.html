Online Movie Recomendation System

Writing the title of your favourite movie in the box below and pressing Enter, my recommendation engine will recommend you 10 movies and give you a score of the likelihood you will like them.

This Web App was realised using Python Flask and PythonAnywhere. The database used to train this recommendation engine was a collection of the top 250 rated movies from IMDB. This recommendation system uses Cosine Similariy in order to make its predictions and the data preprocessing has been handled using Natural Language Processing (NLP).
